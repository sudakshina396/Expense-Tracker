# Expense Tracker Server (Express + TypeScript)

## Env vars
- `PORT` (default: 5000)
- `MONGODB_URI` (default: mongodb://127.0.0.1:27017/expense-tracker)
- `CLIENT_ORIGIN` (default: http://localhost:5173)

## Run
From `server/`:
- `npm i`
- `npm run dev`

Health check:
- GET http://localhost:5000/health
